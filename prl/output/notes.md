## File description

#### Prefix:
 - **H-** for Holmes Burster model.
 - **HR-** for Hindmarsh Rose model.
 - **FN-** for FitzHugh-Nagumo model.

#### Sufix:
 - **-full** for hole parameter space:
    - Horizontal: V<sup>th</sup><sub>KS</sub> &#x2208; [-32.0, -19.0]
    - Vertical I<sub>ext</sub> &#x2208; [34.5, 36.5]
